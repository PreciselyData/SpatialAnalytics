## Setup Google CloudShell

Login to Google Cloud Console

Ensure you are in the right project

Open Cloud Shell by clicking on the icon on the tool bar

Verify the following utilities,

```
gcloud -h
```
```
kubectl -h
```
```
helm -h
```
